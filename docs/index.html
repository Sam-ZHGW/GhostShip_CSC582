A work of interactive fiction.

The Ghost Ship was created with Inform and has IFID F5DFF0FF-1F08-4C96-B790-4DA5B3A5EFD6. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
